Διαχείριση των ΕΑ σας

Μέσω του πίνακα ελέγχου, μπορείτε να διαχειριστείτε οικουμενικά όλες τις ΕΑ σας, επιτρέποντάς σας να κάνετε διάφορες ενέργειες όπως την αντιγραφή ή την εξαγωγή μιας ΕΑ.

Επεξεργασία πληροφοριών σχετικά με μια ΕΑ

Για να επεξεργαστείτε τις πληροφορίες που εμφανίζονται σε μια ΕΑ στον πίνακα ελέγχου, κάντε κλικ στο κείμενο που θέλετε να επεξεργαστείτε και εισάγετε το κείμενό σας.

Αντιγραφή μιας ΕΑ

Για να αντιγράψετε μια ΕΑ, η οποία θα μπορούσε να χρησιμοποιηθεί ως πρότυπο για πολλές από τις εργασίες επεξεργασίας σας:
  • Στην προβολή καρτών, τοποθετήστε τις κουκκίδες στο πάνω δεξιά μέρος της κάρτας που θέλετε να αντιγράψετε και κάντε κλικ στο "Αντιγραφή"·
  • Στην προβολή λίστας, κάντε κλικ στο εικονίδιο "Αντιγραφή".

Εξαγωγή μιας ΕΑ

Για να εξάγετε μια ΕΑ (μορφή json) κάντε κλικ στο εικονίδιο εξαγωγής. Με αυτόν τον τρόπο, μπορείτε να μοιραστείτε μια ΕΑ με τους συναδέλφους ή τους συνεργάτες σας, καθώς θα μπορούν να την εισάγουν στο λογισμικό τους για ΕΑ.

Εισαγωγή μιας ΕΑ

Για την εισαγωγή μιας ΕΑ:
  • Στην προβολή καρτών, κάντε κλικ στο συν ("+") που σχετίζεται με την "Εισαγωγή ΕΑ" στην κάρτα δημιουργίας ΕΑ. Θα εμφανιστεί ένα παράθυρο. Επιλέξτε το αρχείο ΕΑ (αρχείο json) που θέλετε να εισάγετε·
  • Στην προβολή λίστας, κάντε κλικ στο εικονίδιο "Εισαγωγή ΕΑ". Θα εμφανιστεί ένα παράθυρο. Επιλέξτε το αρχείο ΕΑ (αρχείο json) που θέλετε να εισάγετε. Ας σημειωθεί ότι οποιαδήποτε άλλη μορφή αρχείου δεν υποστηρίζεται από το λογισμικό ΕΑ και επομένως δεν θα εισαχθεί στο εργαλείο.

Εξαγωγή της αναφοράς ΕΑ

Για την εξαγωγή της αναφοράς ΕΑ:
  • Στην προβολή καρτών, τοποθετήστε τις κουκκίδες ("…") στην επάνω δεξιά γωνία της κάρτας που θέλετε να αντιγράψετε και κάντε κλικ στο "Εμφάνιση ΕΑ" ή "Εμφάνιση σχεδίου δράσης"·
  • Στην προβολή λίστας, κάντε κλικ στο εικονίδιο "Εμφάνιση ΕΑ" ή "Εμφάνιση σχεδίου δράσης".

Εκτύπωση της αναφοράς ΕΑ

Για εκτύπωση της αναφοράς ΕΑ, ακολουθήστε τα βήματα που περιγράφονται στην ενότητα "Εξαγωγή της αναφοράς ΕΑ" και κάντε κλικ στο εικονίδιο εκτύπωσης.

Επεξεργασία της ΕΑ

Πρώτο βήμα για την σωστή διεξαγωγή μιας ΕΑ· η φάση της έκδοσης θα καλύπτεται συνήθως από τα άτομα που είναι υπεύθυνα για την πρακτική εφαρμογή της επεξεργασίας (τμήμα μάρκετινγκ, ομάδα έργου, τμήμα πληροφορικής …).

Δημιουργία μιας ΕΑ

Για τη δημιουργία μιας ΕΑ:
  • Στην προβολή καρτών, κάντε κλικ στο συν ("+") που σχετίζεται με το "Δημιουργία ΕΑ" στην κάρτα δημιουργίας ΕΑ. Κάντε κλικ στο κουμπί "Έναρξη"·
  • Στην προβολή λίστας, κάντε κλικ στο κουμπί "Δημιουργία ΕΑ". Κάντε κλικ στο κουμπί "Έναρξη".
Είναι υποχρεωτικό να συμπληρώσετε όλα τα πεδία για να μπορείτε να επεξεργαστείτε την ΕΑ. Οι πληροφορίες που εισάγονται πρώτα κατά τη δημιουργία μιας ΕΑ μπορούν να τροποποιηθούν ανά πάσα στιγμή στον πίνακα ελέγχου.

Επεξεργαστείτε ένα πεδίο κειμένου

Για να συμπληρώσετε ένα πεδίο κειμένου, κάντε κλικ στην περιοχή κειμένου με γκρι χρώμα κάτω από την ερώτηση που θέλετε να απαντήσετε. Το συμβολικό κράτησης θέσης κείμενο που εμφανίστηκε προηγουμένως σε αυτήν την περιοχή κειμένου αναφέρεται αυτόματα στη στήλη της βάσης γνώσεων. Μπορείτε να μορφοποιήσετε το κείμενο που εισάγετε χρησιμοποιώντας τη γραμμή εργαλείων που εμφανίζεται κάτω από το πεδίο κειμένου. Για να επεξεργαστείτε ένα πεδίο όπου έχει ήδη εισαχθεί κάποιο κείμενο, κάντε κλικ σε αυτό το κείμενο για να ενεργοποιήσετε την επεξεργασία του πεδίου κειμένου.

Προσθέστε σχόλια

Για να δημοσιεύσετε ένα σχόλιο, κάντε κλικ στο "Σχόλιο" και εισάγετε το σχόλιό σας στο πεδίο που εμφανίζεται. Στη συνέχεια, κάντε κλικ στο "Αποστολή" για να το αποθηκεύσετε.

Χρησιμοποιώντας τη βάση γνώσεων

Η βάση γνώσεων περιλαμβάνει τις ουσιώδεις πληροφορίες που χρειάζεστε για να πραγματοποιήσετε μια ΕΑ. Είναι διαθέσιμη ανα πάσα στιγμή κατά την επεξεργασία μιας ΕΑ και φιλτράρεται αυτόματα σύμφωνα με την ερώτηση που απαντάτε. Μπορείτε επίσης να αναζητήσετε με μη αυτόματο τρόπο τη βάση γνώσεων χρησιμοποιώντας τη γραμμή αναζήτησης που σχετίζεται με αυτήν.

Προσθήκη συνημμένων

Για να προσθέσετε ένα συνημμένο, κάντε κλικ στο "Προσθήκη" στην περιοχή συνημμένων που εμφανίζεται κάτω από το μενού περιήγησης στα αριστερά.

Προσθήκη μέτρων

Για την προσθήκη μέτρων μπορείτε είτε:
  • Να προσθέσετε ένα μέτρο μη αυτόματα, κάνοντας κλικ στο "Προσθήκη ενός άδειου μέτρου"·
  • Να προσθέσετε αυτόματα ένα μέτρο που είναι διαθέσιμο στη βάση γνώσεων, κάνοντας κλικ στο συν ("+") που σχετίζεται με αυτό.

Επεξεργασία ενός κινδύνου

Για να επεξεργαστείτε έναν κίνδυνο, πρέπει να εισάγετε ως ετικέτα, στο αντίστοιχο πεδίο, κάθε αντίκτυπο, απειλή, πηγή και μέτρο που συνδέεται με τον εν λόγω κίνδυνο. Όσον αφορά στα μέτρα, μπορείτε να εισαγάγετε μόνο αυτά που προηγουμένως αποθηκεύτηκαν στην ενότητα "Προγραμματισμένα ή υπάρχοντα μέτρα". Αν θέλετε να προσθέσετε ένα νέο μέτρο, θα πρέπει να το κάνετε στην προαναφερθείσα ενότητα. Για να υποδείξετε τη σοβαρότητα και την πιθανότητα κινδύνου, σύρετε τον κέρσορα στο μετρητή στο αντίστοιχο εκτιμώμενο επίπεδο. Πρέπει επίσης να δικαιολογήσετε αυτήν την εκτίμηση σύμφωνα με τα στοιχεία που είχαν καταχωριστεί προηγουμένως.

Επισκόπηση κινδύνου

Η επισκόπηση κινδύνου σας παρέχει έναν γρήγορο και εύκολο τρόπο κατανόησης των κινδύνων που συνδέονται με την επεξεργασία και των παραγόντων που την επηρεάζουν. Μπορείτε να φιλτράρετε την απεικόνιση κάνοντας κλικ στις διάφορες κατηγορίες (αντίκτυπος, απειλή, πηγή, μέτρο) ή τους κινδύνους.

Αξιολόγηση μιας ΕΑ

Το δεύτερο βήμα της ΕΑ· η αξιολόγηση θα διεξαχθεί από τον ΥΠΔ μαζί με τον Υπεύθυνο Ασφαλείας Πληροφοριακών Συστημάτων (CISO). Η αξιολόγηση μπορεί να γίνει σε ολόκληρο το τμήμα ή σε κάθε ερώτηση ενός τμήματος. Υπάρχουν δύο ή τρεις επιλογές αξιολόγησης.

"Προς διόρθωση"

Η επιλογή "προς διόρθωση" σημαίνει ότι οι πληροφορίες που παρέχονται κατά τη διάρκεια της φάσης επεξεργασίας δεν επαρκούν για να μπορέσουν να αξιολογηθούν. Επιλέγοντας αυτή την αξιολόγηση, η ενότητα θα αποσταλεί εκ νέου σε επεξεργασία.

"Βελτιώσιμη"

Η "βελτιωμένη" επιλογή σημαίνει ότι οι πληροφορίες που παρέχονται κατά τη διάρκεια της φάσης επεξεργασίας είναι επαρκείς, αλλά πρέπει να ληφθούν ορισμένες διορθωτικές ενέργειες για να βελτιωθεί η συμμόρφωση ή η ασφάλεια της επεξεργασίας. Επιλέγοντας αυτή την αξιολόγηση, είναι υποχρεωτική η περιγραφή των διορθωτικών ενεργειών που θα προστεθούν αυτόματα στο σχέδιο δράσης.

"Αποδεκτή"

Η "αποδεκτή" επιλογή σημαίνει ότι οι πληροφορίες που παρέχονται κατά τη διάρκεια της φάσης επεξεργασίας είναι επαρκείς και ότι η επεξεργασία προσωπικών δεδομένων είναι σύμφωνη με τον Κανονισμό και προτατεύει άριστα τα προσωπικά δεδομένα.

Γνώμες του ΥΠΔ και των ενδιαφερόμενων

Αυτή η ενότητα επιτρέπει στον ΥΠΔ να δηλώσει εάν η επεξεργασία πρέπει να υλοποιηθεί ή όχι και να το δικαιολογήσει. Το τμήμα αυτό επιτρέπει επίσης να διευκρινιστεί αν έχει αναζητηθεί η γνώμη των ενδιαφερόμενων ή όχι και να δικαιολογηθούν οι λόγοι πίσω από αυτή την επιλογή καθώς και οι απαντήσεις των ενδιαφερόμενων όπου απαιτείται.

Επικύρωση μιας ΕΑ

Το τελικό βήμα της ΕΑ· η επικύρωση θα γίνει από τον υπεύθυνο επεξεργασίας. Η χαρτογράφηση των κινδύνων και το σχέδιο δράσης αποτελούν δύο εργαλεία που τον βοηθούν να κατανοεί γρήγορα τους κινδύνους που συνδέονται με την επεξεργασία του και τις ενέργειες που πρέπει να αναληφθούν για τη βελτίωσή της. Με βάση την ανάλυσή της, είναι διαθέσιμοι τρεις τρόποι επικύρωσης της ΕΑ.

Χαρτογράφηση κινδύνων

Η χαρτογράφηση κινδύνων απεικονίζει την εξέλιξη του κινδύνου σύμφωνα με τις διορθωτικές ενέργειες που πρότεινε ο αξιολογητής σε σύγκριση με την υπάρχουσα κατάσταση.

Σχέδιο δράσης

Το σχέδιο δράσης συγκεντρώνει όλες τα διορθωτικά μέτρα που υποδείχθηκαν κατά τη φάση αξιολόγησης από τον αξιολογητή. Η απεικόνιση στην κορυφή της σελίδας βοηθάει να καταλάβετε γρήγορα ποια είναι τα σημεία που θα βελτιώσουν την επεξεργασία. Για κάθε διορθωτικό μέτρο στο σχέδιο δράσης, μπορείτε να ορίσετε ημερομηνία εφαρμογής και υπεύθυνο για την υλοποίησή του.

Απόρριψη ΕΑ

Αν θεωρείτε ότι η ΕΑ είναι υπερβολικά επικίνδυνη ή ελλιπής, μπορείτε να την απορρίψετε κάνοντας κλικ στο "Απόρριψη ΕΑ". Θα πρέπει αργότερα να εξηγήσετε τον λόγο για τον οποίο την απορρίψατε. Στη συνέχεια, θα μπορείτε είτε να εγκαταλείψετε την επεξεργασία, οπότε η ΕΑ θα αρχειοθετηθεί χωρίς δυνατότητα μεταγενέστερης τροποποιήσης, είτε να στείλετε την ΕΑ ξανά στον αξιολογητή. Εναπόκειται στον αξιολογητή να λάβει υπόψη τις παρατηρήσεις του υπεύθυνου επεξεργασίας προκειμένου να προτείνει τις απαραίτητες αλλαγές στην ΕΑ.

Έγκριση ΕΑ (υπογεγραμμένη)

Μπορείτε να εγκρίνετε μια ΕΑ φορτώνοντας ένα υπογεγραμμένο αντίγραφο της έκθεσης ΕΑ στο λογισμικό. Επιλέξτε το κουμπί "Έγκριση ΕΑ (υπογεγραμμένη)", εκτυπώστε την αναφορά, υπογράψτε την και προσθέστε την στο λογισμικό ΕΑ ως προσάρτημα της εν λόγω ΕΑ.

Έγκριση ΕΑ (άμεση)

Υπάρχει μια απλοοποιημένη διαδικασία έγκρισης. Για να τη χρησιμοποιήσετε, κάντε κλικ στο κουμπί "Έγκριση ΕΑ (άμεση)". Αυτή η επιλογή επικύρωσης θα πρέπει να χρησιμοποιείται για επεξεργασία παρόμοια με εκείνη που έχει ήδη εγκριθεί με υπογεγραμμένη διαδικασία έγκρισης.
