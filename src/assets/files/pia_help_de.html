Verwalten Sie Ihre DSFA

In der Übersicht können Sie zentral alle Ihre DSFA verwalten, indem Sie verschiedene Aktionen ausführen, wie z.B. eine DSFA kopieren oder exportieren.

Bearbeiten der Informationen einer DSFA

Um die in der Übersicht einer DSFA angezeigten Informationen zu bearbeiten, klicken Sie auf den Text, den Sie bearbeiten möchten, und geben Sie Ihren Text ein.

DSFA kopieren

So kopieren Sie eine DSFA, die als Vorlage für mehrere Ihrer Verarbeitungsvorgänge verwendet werden kann:
  • Bewegen Sie in der Kartenansicht die Punkte oben rechts auf der Karte, die Sie kopieren möchten, und klicken Sie auf "Kopieren";
  • Klicken Sie in der Listenansicht auf das Symbol "Kopieren".

DSFA exportieren

Um eine DSFA (als JSON-Datei) zu exportieren, klicken Sie auf das Exportsymbol. Auf diese Weise können Sie eine DSFA mit Ihren Kollegen oder Partnern teilen, da diese die Datei in ihr PIA-Tool importieren können.

DSFA importieren

So importieren Sie eine DSFA:
  • Klicken Sie in der Kartenansicht auf das Plus ("+"), das mit "DSFA importieren" auf der DSFA-Erstellungskarte verknüpft ist. Ein Fenster wird angezeigt. Wählen Sie die DSFA (als JSON-Datei) aus, die Sie importieren möchten.
  • Klicken Sie in der Listenansicht auf das Symbol "DSFA importieren". Ein Fenster wird angezeigt. Wählen Sie die DSFA (als JSON-Datei) aus, die Sie importieren möchten. Beachten Sie, dass andere Dateiformate vom PIA-Tool nicht unterstützt werden und daher nicht in das Tool importiert werden.

DSFA Bericht exportieren

So exportieren Sie den DSFA-Bericht:
  • Bewegen Sie in der Kartenansicht die Punkte ("...") oben rechts auf der Karte und klicken Sie auf "DSFA anzeigen" oder "Aktionsplan anzeigen";
  • Klicken Sie in der Listenansicht auf das Symbol "DSFA anzeigen" oder "Aktionsplan anzeigen".

DSFA Bericht drucken

Um den DSFA-Bericht zu drucken, folgen Sie den unter "Exportieren des DSFA-Berichts" beschriebenen Schritten und klicken Sie auf das Drucken-Symbol.

DSFA bearbeiten

Erster Schritt zur ordnungsgemäßen Durchführung einer DSFA. Die Bearbeitungsphase wird in der Regel von den Verantwortlichen für die praktische Durchführung der Verarbeitung (Marketingabteilung, Projektteam, IT-Abteilung ...) ausgefüllt.

DSFA anlegen

So erstellen Sie eine DSFA:
  • Klicken Sie in der Kartenansicht auf das Plus ("+"), das mit "DSFA anlegen" auf der DSFA-Erstellungskarte verknüpft ist. Klicken Sie auf die Schaltfläche "Starten";
  • Klicken Sie in der Listenansicht auf die Schaltfläche "DSFA anlegen". Klicken Sie auf die Schaltfläche "Start".
Sie müssen alle Felder ausfüllen, um die DSFA bearbeiten zu können. Die Informationen, die beim Erstellen einer DSFA eingegeben wurden, können jederzeit in der Übersicht bearbeitet werden.

Ein Textfeld bearbeiten

Um ein Textfeld zu füllen, klicken Sie auf den grauen Textbereich unter der Frage, die Sie beantworten möchten. Der Platzhaltertext, der zuvor in diesem Textbereich angezeigt wurde, wird automatisch in der Wissensbasis-Spalte angezeigt. Sie können den eingegebenen Text formatieren, indem Sie die Symbolleiste unter dem Textfeld verwenden. Um ein Feld zu bearbeiten, in dem bereits Text eingegeben wurde, klicken Sie auf diesen Text, um das Textfeld zu aktivieren.

Kommentare hinzufügen

Um einen Kommentar hinzufügen, klicken Sie auf "Kommentieren" und geben Sie Ihren Kommentar in das Feld ein, das angezeigt wird. Klicken Sie anschließend auf "Senden", um ihn zu speichern.

Die Wissensbasis nutzen

Die Wissensbasis enthält die wesentlichen Informationen, die Sie für die Durchführung einer DSFA benötigen. Sie ist während der Bearbeitung einer DSFA jederzeit verfügbar und wird automatisch entsprechend der von Ihnen beantworteten Frage gefiltert. Sie können die Wissensbasis auch manuell über die zugehörige Suchleiste durchsuchen.

Anlagen hinzufügen

Um eine Anlage hinzuzufügen, klicken Sie auf "Hinzufügen" im Bereich "Anlagen", die unter dem Navigationsmenü auf der linken Seite angezeigt werden.

Maßnahmen hinzufügen

Um eine Maßnahme hinzuzufügen, können Sie entweder:
  • Manuell eine neue Maßnahme hinzufügen, indem Sie auf "Erstellen Sie eine Maßnahme ..." klicken oder
  • eine in der Wissensdatenbank verfügbare Maßnahme automatisch hinzufügen, indem Sie auf die dazugehörige Schaltfläche plus ("+") klicken.
  • Anschließend können Sie die konkret getroffenen Maßnahmen in das Textfeld eingeben.

Risiken bearbeiten

Um ein Risiko zu bearbeiten, müssen Sie im entsprechenden Feld die Auswirkungen, Bedrohungen, Risikoquellen und Maßnahmen, die mit dem betrachteten Risiko verknüpft sind, aus der angebotenen Liste auswählen. Als Maßnahmen können Sie nur diejenigen eingeben, die zuvor im Abschnitt "Geplante oder vorhandene Maßnahmen" gespeichert wurden. Wenn Sie eine neue Maßnahme hinzufügen möchten, sollten Sie dies im oben genannten Abschnitt tun. Um den Risikoschweregrad und die Eintrittswahrscheinlichkeit anzugeben, bewegen Sie den Zeiger der Skala auf den angenommenen Wert. Sie müssen diese Einschätzung anhand der zuvor gemachten Angaben begründen.

Risikoübersicht

Die Risikoübersicht bietet Ihnen einen schnellen und einfachen Weg, die mit der Verarbeitung und den Einflussfaktoren verbundenen Risiken zu verstehen. Sie können die Darstellung filtern, indem Sie auf die verschiedenen Kategorien (Auswirkung, Bedrohung, Quelle, Maßnahme) oder Risiken klicken.

DSFA überpüfen

Der zweite Schritt der DSFA, die Überprüfung, wird z.B. vom DSB gemeinsam mit dem CISO durchgeführt. Die Überprüfung kann für einen ganzen Abschnitt oder jede Frage eines Abschnitts vorgenommen werden. Entweder sind zwei oder drei Bewertungsoptionen verfügbar.

"Zu berichtigen"

Die Bewertung "zu berichtigen" bedeutet, dass die während der Bearbeitungsphase bereitgestellten Informationen nicht ausreichend für eine Bewertung sind. Es ist dann keine Bestätigung möglich. Zur Verbesserung der DSFA muss eine Kopie angelegt und vom Bearbeiter ergänzt werden. Die kopierte und kompletierte DSFA kann dann erneut zur Überprüfung vorgelegt werden.

"Verbesserungsfähig"

Die Bewertung "verbesserungsfähig" bedeutet, dass die in der Bearbeitungsphase bereitgestellten Informationen ausreichend sind, jedoch noch einige Korrekturmaßnahmen ergriffen werden müssen, die Datenschutzkonformität bzw. die Sicherheit der Verarbeitung zu verbessern. Wenn Sie diese Bewertung auswählen, müssen Sie zudem die erforderlichen Korrekturmaßnahmen beschreiben, die dann automatisch zum Aktionsplan hinzugefügt werden.

"Akzeptabel"

Die Bewertung "akzeptabel" bedeutet, dass die in der Bearbeitungsphase bereitgestellten Informationen ausreichend sind, die Verarbeitung datenschutzkonform geschieht und die personenbezogenen Daten ausreichend geschützt sind.

Stellungnahmen des DSBs und der Betroffenen

In diesem Abschnitt kann der DSB abgeben, ob die Verarbeitung implementiert werden soll oder nicht und dies begründen. In diesem Abschnitt kann auch angegeben werden, ob die Meinung der betroffenen Personen eingeholt wurde oder nicht, sowie die Gründe für diese Entscheidung und die abgegebenen Antworten der betroffenen Personen.

DSFA bestätigen

Letzter Schritt der DSFA ist die Bestätigung durch den Verantwortlichen. Der Risikoüberblick und die Risikokartierung sind zwei Instrumente, die ihm helfen, die mit seiner Verarbeitung verbundenen Risiken und die ergriffenen bzw. zu ergreifenden Maßnahmen zu deren Minderung zu verstehen. Abhängig von seiner Analyse hat er drei Möglichkeiten, die DSFA zu bestätigen.

Risikokartierung

Die Risikokartierung stellt die Risikoentwicklung gemäß den von den Prüfern vorgeschlagenen Korrekturmaßnahmen im Vergleich zur bestehenden Situation dar.

Aktionsplan

Der Aktionsplan fasst alle Korrekturmaßnahmen zusammen, die während der Überprüfung von den Prüfern angegeben wurden. Die Darstellung auf der Seite hilft schnell zu verstehen, welche Punkte in der Verarbeitung zu verbessern sind. Für jede Korrekturmaßnahme im Aktionsplan können Sie ein Fälligkeitsdatum und eine verantwortliche Person für die Umsetzung festlegen.

DSFA ablehnen

Wenn Sie der Ansicht sind, dass die DSFA zu riskant oder unvollständig ist, können Sie sie ablehnen, indem Sie auf "DSFA ablehnen" klicken. Sie sollten anschließend den Grund für die Ablehnung erklären. Sie können dann entweder die Verarbeitung abbrechen, in diesem Fall wird die DSFA archiviert, ohne dass sie später bearbeitet werden kann, oder Sie können sie an die Prüfer zurücksenden. Diese sollten die Kommentare des für die Verarbeitung Verantwortlichen berücksichtigen und die erforderlichen Änderungen in der DSFA vorschlagen.

DSFA bestätigen (mit Unterschrift)

Sie können eine DSFA bestätigen, indem Sie eine unterschriebene Kopie des DSFA-Berichts in das Tool hochladen. Wählen Sie den Button "Unterschriebene Bestätigung", drucken Sie den Bericht aus, lassen Sie ihn unterschreiben und fügen Sie ihn als Anlage der betrachteten DSFA im PIA-Tool hinzu.

DSFA bestätigen (einfach)

Es gibt ein vereinfachtes Genehmigungsverfahren. Um es zu verwenden, klicken Sie auf die Schaltfläche "Einfache Bestätigung". Diese Bestätigungssoption sollte für Verarbeitungen verwendet werden, bei denen ein ähnliches Verfahren bereits per Unterschrift genehmigt wurde.
