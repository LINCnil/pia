Festlegung des Titels der DSFA

Festlegung von Rollen

    Für die Durchführung einer DSFA sind drei Rollen zu definieren:

  • Bearbeiter:
    Der Bearbeiter stammt in der Regel aus einem Bereich, der mit der praktischen Datenverarbeitung betraut ist (Fachabteilung, Projektteam, IT-Abteilung). Er legt die DSFA an und gibt die erforderlichen Angaben in das Tool ein. Hat er alle Felder eines Abschnitts komplett ausgefüllt, kann er die Überprüfung durch den Prüfer anfordern (andernfalls wird diese Option nicht angeboten). Hierbei kann entweder abschnittweise vorgegangen werden, oder er füllt die DSFA zunächst komplett aus.
    Achtung: Es empfiehlt sich, zunächst die DSFA komplett auszufüllen, bevor eine Überprüfung angefordert wird, da ansonsten spätere Ergänzungen nicht möglich sind. Dies betrifft insbesondere den Bereich Risiken, da dort keine weiteren Maßnahmen ergänzt werden können, wenn zu früh eine Überprüfung angefordert wurde.

  • Prüfer:
    Der Prüfer sollte über Kenntnisse zum Datenschutz und zur IT-Sicherheit verfügen (z.B. DSB, CISO). Er überprüft die Angaben und entscheidet, ob Korrekturen oder Verbesserungen nötig sind.
    Anm. Ist eine Korrektur erforderlich, so bietet das Tool in der aktuellen Version die Funktionalitäten zur Bestätigung nicht an. Es muss eine Kopie angelegt und vom Bearbeiter ergänzt werden.
    Erst wenn der Prüfer seine Bewertung in „verbesserungsfähig“ oder „akzeptiert“ umwandelt, ist eine Bestätigung möglich. Sieht er Verbesserungsbedarf, muss er genauere Korrekturmaßnahmen für den Aktionsplan angeben. Zudem erfolgt in diesem Schritt die Beteiligung des behördlichen / betrieblichen Datenschutzbeauftragten sowie der Betroffenen.

  • Bestätigender:
    Dieser bestätigt im Namen des Verantwortlichen, dass die DSFA den Anforderungen genügt und somit das erforderliche Datenschutzniveau erreicht wurde. Danach ist die DSFA abgeschlossen und kann im Rahmen des Datenschutzmanagements dokumentiert werden.

Verwalten Sie Ihre DSFAs

In der Übersicht können Sie zentral alle Ihre DSFAs verwalten, indem Sie verschiedene Aktionen ausführen, wie z.B. eine DSFA duplizieren oder exportieren.

Bearbeiten der Informationen einer DSFA

Um die in der Übersicht einer DSFA angezeigten Informationen zu bearbeiten, klicken Sie auf den Text, den Sie bearbeiten möchten, und geben Sie Ihren Text ein.

DSFA duplizieren

So duplizieren Sie eine DSFA, die als Vorlage für mehrere Ihrer Verarbeitungsvorgänge verwendet werden kann:
  • Bewegen Sie in der Kartenansicht die Punkte oben rechts auf der Karte, die Sie duplizieren möchten, und klicken Sie auf "Duplizieren";
  • Klicken Sie in der Listenansicht auf das Symbol "Duplizieren".

DSFA exportieren

Um eine DSFA (als JSON-Datei) zu exportieren, klicken Sie auf das Exportsymbol. Auf diese Weise können Sie eine DSFA mit Ihren Kollegen oder Partnern teilen, da diese die Datei in ihr PIA-Tool importieren können.

DSFA importieren

So importieren Sie eine DSFA:
  • Klicken Sie in der Kartenansicht auf das Plus ("+"), das mit "DSFA importieren" auf der DSFA-Erstellungskarte verknüpft ist. Ein Fenster wird angezeigt. Wählen Sie die DSFA (als JSON-Datei) aus, die Sie importieren möchten.
  • Klicken Sie in der Listenansicht auf das Symbol "DSFA importieren". Ein Fenster wird angezeigt. Wählen Sie die DSFA (als JSON-Datei) aus, die Sie importieren möchten. Beachten Sie, dass andere Dateiformate vom PIA-Tool nicht unterstützt werden und daher nicht in das Tool importiert werden.

DSFA Bericht exportieren

So exportieren Sie den DSFA-Bericht:
  • Bewegen Sie in der Kartenansicht die Punkte ("...") oben rechts auf der Karte und klicken Sie auf "DSFA anzeigen" oder "Aktionsplan anzeigen";
  • Klicken Sie in der Listenansicht auf das Symbol "DSFA anzeigen" oder "Aktionsplan anzeigen".

DSFA Bericht drucken

Um den DSFA-Bericht zu drucken, folgen Sie den unter "Exportieren des DSFA-Berichts" beschriebenen Schritten und klicken Sie auf das Drucken-Symbol.

DSFA bearbeiten

Erster Schritt zur ordnungsgemäßen Durchführung einer DSFA. Die Bearbeitungsphase wird in der Regel von den Verantwortlichen für die praktische Durchführung der Verarbeitung (Marketingabteilung, Projektteam, IT-Abteilung ...) ausgefüllt.

DSFA anlegen

So erstellen Sie eine DSFA:
  • Klicken Sie in der Kartenansicht auf das Plus ("+"), das mit "DSFA anlegen" auf der DSFA-Erstellungskarte verknüpft ist. Klicken Sie auf die Schaltfläche "Starten";
  • Klicken Sie in der Listenansicht auf die Schaltfläche "DSFA anlegen". Klicken Sie auf die Schaltfläche "Starten".
Sie müssen alle Felder ausfüllen, um die DSFA bearbeiten zu können. Die Informationen, die beim Erstellen einer DSFA eingegeben wurden, können jederzeit in der Übersicht bearbeitet werden. |

Ein Textfeld bearbeiten

Um ein Textfeld zu füllen, klicken Sie auf den grauen Textbereich unter der Frage, die Sie beantworten möchten. Der Platzhaltertext, der zuvor in diesem Textbereich angezeigt wurde, wird automatisch in der Wissenbasis-Spalte angezeigt. Sie können den eingegebenen Text formatieren, indem Sie die Symbolleiste unter dem Textfeld verwenden. Um ein Feld zu bearbeiten, in dem bereits Text eingegeben wurde, klicken Sie auf diesen Text, um die Ausgabe des Textfelds zu aktivieren.

Kommentare hinzufügen

Um einen Kommentar hinzufügen, klicken Sie auf "Kommentieren" und geben Sie Ihren Kommentar in das Feld ein, das angezeigt wird. Klicken Sie anschließend auf "Senden", um ihn zu speichern.

Die Wissensbasis nutzen

Die Wissensbasis enthält die wesentlichen Informationen, die Sie für die Durchführung einer DSFA benötigen. Sie ist während der Bearbeitung einer DSFA jederzeit verfügbar und wird automatisch entsprechend der von Ihnen beantworteten Frage gefiltert. Sie können die Wissensbasis auch manuell über die zugehörige Suchleiste durchsuchen.

Anhänge hinzufügen

Um einen Anhang hinzuzufügen, klicken Sie auf "Hinzufügen" unter den Anhängen, die unter dem Navigationsmenü auf der linken Seite angezeigt werden.

Maßnahmen hinzufügen

Um eine Maßnahme hinzuzufügen, können Sie entweder:
  • Manuell eine neue Maßnahme hinzufügen, indem Sie auf "Erstellen Sie eine Maßnahme ..." klicken oder
  • eine in der Wissensdatenbank verfügbare Maßnahme automatisch hinzufügen, indem Sie auf die dazugehörige Schaltfläche plus ("+") klicken.

Risiken bearbeiten

Um ein Risiko zu bearbeiten, müssen Sie im entsprechenden Feld jede Auswirkung, Bedrohung, Quelle und Maßnahme, die mit dem betrachteten Risiko verknüpft sind, als Tag eingeben. Als Maßnahmen können Sie nur diejenigen eingeben, die zuvor im Abschnitt "Geplante oder vorhandene Maßnahmen" gespeichert wurden. Wenn Sie eine neue Maßnahme hinzufügen möchten, sollten Sie dies im oben genannten Abschnitt tun. Um den Risikoschweregrad und die Eintrittswahrscheinlichkeit anzugeben, bewegen Sie den Zeiger auf der Skala auf die entsprechende geschätzte Ebene. Sie müssen diese Schätzung auch anhand der zuvor gemachten Angaben begründen.

Risikoübersicht

Die Risikoübersicht ermöglicht Ihnen einen schnellen und einfachen Weg, die mit der Verarbeitung und den Einflussfaktoren verbundenen Risiken zu verstehen. Sie können die Visualisierung filtern, indem Sie auf die verschiedenen Kategorien (Auswirkung, Bedrohung, Quelle, Maßnahme) oder Risiken klicken.

DSFA bewerten

Zweiter Schritt der DSFA, die Evaluierung wird vom DSB gemeinsam mit dem CISO durchgeführt. Die Bewertung kann an einem ganzen Abschnitt oder an jeder Frage eines Abschnitts vorgenommen werden. Entweder sind zwei oder drei Bewertungsoptionen verfügbar.

"Zu berichtigen"

Die Bewertung "zu berichtigen" bedeutet, dass die während der Bearbeitungsphase bereitgestellten Informationen nicht ausreichend sind, um ausgewertet werden zu können. Wenn Sie diese Bewertung auswählen, wird der Abschnitt zur weiteren Bearbeitung zurückgeschickt.

"Verbesserungsfähig"

Die Bewertung "verbesserungsfähig" bedeutet, dass die in der Bearbeitungsphase bereitgestellten Informationen ausreichend sind, jedoch einige Korrekturmaßnahmen ergriffen werden müssen, um die Konformität oder Sicherheit der Verarbeitung zu verbessern. Wenn Sie diese Bewertung auswählen, müssen Sie unbedingt die Korrekturmaßnahmen beschreiben, die dann automatisch zum Aktionsplan hinzugefügt werden.

"Tragbar"

Die Bewertung "tragbar" bedeutet, dass die in der Bearbeitungsphase bereitgestellten Informationen ausreichend sind und dass die Verarbeitung datenschutzkonform geschieht und die personenbezogenen Daten optimal schützt.

Stellungnahmen des DSBs und der Betroffenen

In diesem Abschnitt kann der DSB angeben, ob die Verarbeitung implementiert werden soll oder nicht und dies begründen. In diesem Abschnitt kann auch angegeben werden, ob die Meinung der betroffenen Personen eingeholt wurde oder nicht, sowie die Gründe für diese Entscheidung und die abgegebenen Antworten der betroffenen Personen darlegt werden.

DSFA bestätigen

Letzter Schritt der DSFA ist die Bestätigung durch den Verantwortlichen. Der Risikoüberblick und die Risikokartierung sind zwei Instrumente, die ihm helfen, die mit seiner Verarbeitung verbundenen Risiken schnell zu verstehen und welche Maßnahmen zu deren Minderung ergriffen werden müssen. Abhängig von seiner Analyse hat er drei Möglichkeiten, die DSFA zu bestätigen.

Risikokartierung

Die Risikokartierung visualisiert die Risikoentwicklung gemäß den von den Prüfern vorgeschlagenen Korrekturmaßnahmen im Vergleich zur bestehenden Situation.

Aktionsplan

Der Aktionsplan fasst alle Korrekturmaßnahmen, die während der Bewertungsphase von den Prüfern angegeben wurden, zusammen. Die Visualisierung oben auf der Seite hilft schnell zu verstehen, welche Punkte in der Verarbeitung zu verbessern sind. Für jede Korrekturmaßnahme im Aktionsplan können Sie ein Fälligkeitsdatum und eine verantwortliche Person für die Umsetzung festlegen.

DSFA zurückweisen

Wenn Sie der Ansicht sind, dass die DSFA zu riskant oder unvollständig ist, können Sie sie ablehnen, indem Sie auf "DSFA zurückweisen" klicken. Sie sollten anschließend den Grund für die Ablehnung erklären. Sie können dann entweder die Verarbeitung abbrechen, in diesem Fall wird die DSFA archiviert, ohne dass sie später bearbeitet werden kann, oder Sie können sie an die Prüfer zurücksenden. Sie werden die Kommentare des für die Verarbeitung Verantwortlichen berücksichtigen müssen, um die erforderlichen Änderungen in der DSFA vorzuschlagen.

DSFA bestätigen (mit Unterschrift)

Sie können eine DSFA bestätigen, indem Sie eine unterschriebene Kopie des DSFA-Berichts mit der Software hochladen. Wählen Sie den Button "Lassen Sie den Verantwortlichen die DSFA unterzeichnen", drucken Sie den Bericht aus, lassen Sie ihn unterschreiben und fügen Sie ihn als Anhang der betrachteten DSFA im PIA-Tool hinzu.

DSFA bestätigen (einfach)

Es gibt ein vereinfachtes Genehmigungsverfahren. Um es zu verwenden, klicken Sie auf die Schaltfläche "DSFA (direkt) genehmigen". Diese Bestätigungssoption sollte für eine Verarbeitung verwendet werden, die denen ähnelt, die bereits mit einem unterzeichneten Genehmigungsprozess genehmigt wurden.
